YouTube OAuth 設定ガイド

twitchTransFreeNeo で YouTube Live チャットに翻訳を投稿するための設定方法

このページについて

twitchTransFreeNeoでYouTube Liveのチャットに翻訳を投稿するには、YouTube Data API v3のOAuth認証が必要です。このガイドでは、必要な「Client ID」と「Client Secret」の取得方法を説明します。

所要時間: 約10〜15分

認証時のアカウント確認について

認証時は、現在Googleにログインしているアカウントで認証されます。

複数のGoogleアカウントをお持ちの場合は、認証前に正しいアカウントでログインしているか確認してください。

twitchTransFreeNeoの「YouTube認証を行う」ボタンを押すと、ブラウザが開き、認証するGoogleアカウントを選択できます。認証完了後、アプリ内に認証されたアカウント名が表示されます。

読み取り専用モードについて

OAuth認証を設定しなくても、YouTube Liveのチャットを読み取って翻訳表示することは可能です。チャットへの翻訳投稿が不要な場合は、この設定は不要です。

事前準備

  • Googleアカウント（YouTubeで使用しているもの）
  • YouTube Liveで配信できる状態のチャンネル

設定手順

1 Google Cloud Console にアクセス

以下のボタンをクリックして、Google Cloud Console を開きます。

Google Cloud Console を開く

初めての場合は、利用規約への同意が必要です。

2 新しいプロジェクトを作成
  1. 画面上部のプロジェクト選択ドロップダウンをクリック
  2. 「新しいプロジェクト」をクリック
  3. プロジェクト名を入力（例: twitchTransFreeNeo）
  4. 「作成」をクリック
[プロジェクト作成画面のスクリーンショット]
3 YouTube Data API v3 を有効化

以下のボタンをクリックして、YouTube Data API v3 の有効化ページを開きます。

YouTube Data API v3 を有効化
  1. 作成したプロジェクトが選択されていることを確認
  2. 「有効にする」ボタンをクリック
4 OAuth 同意画面を設定
OAuth 同意画面を設定
  1. User Type で「外部」を選択し、「作成」をクリック
  2. アプリ情報を入力:
    • アプリ名: twitchTransFreeNeo
    • ユーザーサポートメール: 自分のメールアドレス
    • デベロッパーの連絡先: 自分のメールアドレス
  3. 「保存して次へ」をクリック
  4. スコープの設定は変更せず「保存して次へ」
  5. テストユーザーに自分のGoogleアカウントを追加
  6. 「保存して次へ」で完了

テストユーザーについて

「外部」で作成したアプリは、テストユーザーとして追加されたGoogleアカウントのみが使用できます。自分のアカウントを必ず追加してください。

5 OAuth クライアント ID を作成
認証情報ページを開く
  1. 「+ 認証情報を作成」をクリック
  2. 「OAuth クライアント ID」を選択
  3. アプリケーションの種類: 「デスクトップアプリ」を選択
  4. 名前を入力（例: twitchTransFreeNeo Desktop）
  5. 「作成」をクリック
6 Client ID と Client Secret をコピー

作成が完了すると、以下の情報が表示されます：

  • クライアント ID - 長い文字列（.apps.googleusercontent.com で終わる）
  • クライアント シークレット - 短い英数字の文字列

これらを twitchTransFreeNeo の設定画面にコピー＆ペーストしてください。

設定完了！

Client ID と Client Secret を twitchTransFreeNeo の「設定」→「基本設定」→「YouTube Live設定」に入力し、「YouTube認証を行う」ボタンをクリックしてください。

twitchTransFreeNeo での設定

  1. twitchTransFreeNeo を起動
  2. 「設定」ボタンをクリック
  3. 「基本設定」タブを選択
  4. プラットフォームで「YouTube Live のみ」または「同時配信」を選択
  5. 「YouTube Client ID」に取得した Client ID を入力
  6. 「YouTube Client Secret」に取得した Client Secret を入力
  7. 「YouTube認証を行う」ボタンをクリック
  8. ブラウザが開くので、Googleアカウントでログインして許可
  9. 「認証済み」と表示されれば完了！

よくある質問（FAQ）

Q: 「このアプリは確認されていません」と表示されます

A: 自分で作成したアプリなので正常です。「詳細」→「（アプリ名）に移動（安全ではないページ）」をクリックして続行してください。

Q: 認証に失敗します

A: 以下を確認してください：

  • OAuth同意画面でテストユーザーに自分を追加したか
  • YouTube Data API v3が有効になっているか
  • Client IDとSecretが正しくコピーされているか

Q: クォータ（quota）エラーが出ます

A: YouTube Data API には1日あたりの使用量制限があります。翻訳投稿の頻度が高すぎる場合に発生することがあります。しばらく待ってから再試行してください。

Q: 投稿せずに翻訳表示だけしたい

A: OAuth設定は不要です。動画IDだけ入力すれば、チャットの読み取りと翻訳表示が可能です。また、「表示のみモード」をオンにすることでも投稿を無効にできます。
